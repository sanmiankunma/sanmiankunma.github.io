建站日志

2018-04-22

闲来无事心血来潮搭建了这个博客，用的是 Hexo + GitHub 的方法。作为小白，完全依靠网上的教程操作完成，过程不算太复杂，其实还挺有趣。

在页面效果上，虽说自定义程度高，但由于不具备代码相关知识，对于一些想修改的地方只能慢慢琢磨和尝试，并依旧是借助网络的力量。

不过说到底，我也并没有什么东西可写，至少目前没有，所以这博客空荡荡如你所见，但对主题的小修小改却又莫名感兴趣，以致于最后似乎重点和乐趣不在写内容上，全在折腾主题上了。

所以本站将长期处于测试和优化阶段……
